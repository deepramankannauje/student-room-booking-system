# Student Room Booking System

A full-stack room booking application built with React, Node.js, Express, and Supabase.

## Features

### Student
- User Registration & Login
- Browse Rooms
- Book Rooms
- View Booking History
- Profile Management

### Admin
- Dashboard
- Create Rooms
- Manage Bookings
- Approve/Reject Bookings
- Cancel & Complete Bookings

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Axios
- React Hook Form
- Zod

Backend
- Node.js
- Express.js
- JWT Authentication
- Supabase

Database
- PostgreSQL (Supabase)
